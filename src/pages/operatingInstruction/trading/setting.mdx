## 2. 设置 python 解析器和虚拟环境

在策略编辑页面中，通过点击右下角的“设置”图标对 python 解析器和虚拟环境进行设置。

![](http://www.myquant.cn/uploads/202111/attach_16b3f3d0c4b7cbab.png)

---

## 3. 打开策略文件

在策略编辑页面中，点击右下角的“设置”图标，在“运行文件”一行选择“打开”。

![](http://www.myquant.cn/uploads/202111/attach_16b3f3d64ce1082e.png)

---

## 4. 修改策略目录

在策略编辑页面中，点击右下角的“设置”图标，在“策略目标”一行选择“修改”。

![](http://www.myquant.cn/uploads/202111/attach_16b3f3fbeed5330d.png)
