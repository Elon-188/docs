## 1. 终端简介

掘金量化终端是一款为专业量化投资打造的功能齐备的落地式终端，集成了策略开发到实盘的模块化功能，打通研究、仿真和绩效链路、兼容多种编程语言，易于使用、性能可靠，能够帮助量化投资者提高策略开发效率、减少 IT 投入。

掘金量化终端目前仅支持在 Windows 环境下使用，为保障运行稳定，推荐至少使用以下配置：

## 2. 安装步骤

### **1. 下载、安装终端**

访问掘金官网：[http://www.myquant.cn](http://www.myquant.cn/.html)，点击页面中的“免费下载”按钮，下载安装包后进行安装。

![](http://www.myquant.cn/uploads/202111/attach_16b396b4970895f7.png)

### **2. 注册/登录账号**

安装完成后，打开掘金终端进行登录。还未注册账号者，可点击“申请账号”前往注册。

![](http://www.myquant.cn/uploads/202207/attach_16fee1c78d27647f.png)

> “自动登录”可用于快速登录；

> “通讯设置”用于选择不同行情节点；

> “状态报告”用于检查终端的必要网络环境，登录异常时可查看具体错误。

### **3. 安装编程语言**

目前，掘金支持 Python、C++、C#、Matlab 四种编程语言，用户可自行安装熟悉的编程语言。如已安装者，可跳过此步骤。

使用的 Python 的用户需注意，掘金目前仅支持`3.6.5以上，3.7.*，3.8.*和3.9.*`的版本。您可以点击[www.python.org](http://www.python.org/.html)前往安装。

### **4. 安装 SDK**

登录进入终端后，点击页面顶部的“量化研究”，选择“SDK 下载”，用户可点击对应的编程语言，安装相应的 SDK。

![](http://www.myquant.cn/uploads/202111/attach_16b39771294d3676.png)

SDK 安装成功后，刷新该页面将显示已安装版本，代表安装成功。

![](http://www.myquant.cn/uploads/202111/attach_16b3977a2fa68616.png)

### **若未自动安装成功**，请按照下方提示手动安装。

1、使用快捷键“win+R”打开电脑的”运行”，输入“CMD”

![](http://www.myquant.cn/uploads/202111/attach_16b397a16d7fc40f.png)

2、回到终端页面，点击“SDK 下载”页面中的“复制”按钮。

![](http://www.myquant.cn/uploads/202111/attach_16b397bbbd58b2f1.png)

3、使用 Ctrl+V 粘贴代码到框中，然后按回车键。页面如下，表示已经安装完成。此时回到终端页面确认安装结果即可。

![](http://www.myquant.cn/uploads/202111/attach_16b397c202c976c3.png)

以上就是关于安装终端的全部内容，如有其它与“安装”相关的疑问，可前往“常见问题”中查看或直接检索关键词。
