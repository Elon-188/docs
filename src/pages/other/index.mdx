## 4. Python SDK 下载

### Python SDK 说明

掘金 Python 语言版本 API 策略开发包。开发包内含 API，文档，示例策略[查看](https://www.myquant.cn/docs/python.html)

- 支持`2.7.*， 3.6.*，3.7.*，3.8.*和3.9.*`的 Python 版本

- 支持 Windows 和 Linux 系统的 Python

- 支持 64 位和 32 位的 Python

### 安装下载方式

- 掘金 Python SDK 名为 gm， 已经打包上传到 PYPI 库

- 可以通过命令行`python -m pip install gm -i https://pypi.doubanio.com/simple`快速下载安装
